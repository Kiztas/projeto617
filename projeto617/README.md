## Descrição

Este projeto foi desenvolvido em grupo no âmbito da UC00617.

Consiste na criação de um pequeno site em HTML, com o objetivo de praticar o uso de Git e GitHub, incluindo a criação de branches, realização de commits, utilização de Pull Requests e resolução de conflitos.

* membro-isaac.html — página pessoal do Isaac Fernandes.

## Trabalho realizado

Durante o desenvolvimento do projeto, a equipa realizou as seguintes tarefas:

* Criação das páginas pessoais de cada elemento
* Construção conjunta da página principal (index.html)
* Utilização de branches para desenvolvimento individual
* Realização de commits frequentes e organizados
* Abertura e revisão de Pull Requests
* Resolução do conflito do lema antes da entrega final

## Tecnologias usadas

* HTML
* CSS (fornecido pelo formador)
* Git
* GitHub
* GitHub Pages

## Organização do trabalho

O trabalho foi organizado da seguinte forma:

* Cada página pessoal foi desenvolvida num branch próprio pelo respetivo membro
* O index.html foi construído em sessões de trabalho colaborativo
* Todos os Pull Requests foram revistos por colegas e aprovados pelo dono do repositório (Ariano)
* Nenhum elemento fez merge do seu próprio Pull Request;
* O conflito obrigatório foi resolvido antes da conclusão do projeto.
